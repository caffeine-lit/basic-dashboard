# Session 23 Basic Dashboard

Deploy-ready Streamlit app package extracted from this homework repo.

## Run locally

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Deploy on Streamlit Community Cloud

1. Create a new GitHub repository (in your other account).
2. Upload all files from this folder to that repo root.
3. In Streamlit Community Cloud, create a new app and select:
: Main file path: `streamlit_app.py`.
4. Deploy.
